# 2048
My implementation of famous game 2048
*** WIP ***

# Requirements

```
pip install -r requirements.txt
```

# Run

```
python 2048.py
```

# Inputs
- UP : w, i, 8
- LEFT : a, j, 4
- DOWN : s, k, 2
- RIGHT : d, l, 6
